## 概述

Pinlens 是一款强大的 macOS 屏幕区域捕获和实时查看应用，允许你选择屏幕上的任意区域并在独立窗口中实时显示该区域的内容。支持缩放、标尺、网格、辅助线等多种专业辅助工具，是 UI 设计、开发调试、内容创作的得力助手。

## 核心功能

### 🎯 区域选择与实时预览
- 灵活拖拽选择任意屏幕区域
- 支持 8 方向调整大小和移动
- 实时显示选定区域内容
- 多显示器支持
- 窗口置顶功能

### 🔍 缩放与导航
- 支持 0.5x - 10x 缩放
- 鼠标手势缩放
- 实时显示缩放级别和视口位置
- 窗口自动适应
- 滚动浏览完整内容

### 📐 设计辅助工具
- 可开关的水平/垂直标尺
- 自定义辅助线（从标尺拖拽创建）
- 辅助线位置调整和删除
- 可自定义颜色和线宽的网格显示
- 辅助线和网格颜色自定义

### ⚡ 便捷操作
- 轻量级菜单栏应用
- 快捷键快速启动
- ESC 键快速关闭选择覆盖层

## 安装

### 系统要求
- macOS 15.0 (Sonoma) 或更高版本
- 屏幕录制权限

### 快速开始

1. **启动应用** - 应用启动后会在菜单栏显示图标
2. **选择区域** - 点击菜单栏图标选择"选择"，或按 `Cmd+Ctrl+P`
3. **调整选区** - 拖拽创建选区，使用边角和边中间的控制点调整大小
4. **确认选择** - 点击底部控制栏的"确认"按钮
5. **查看内容** - 应用会创建一个新窗口显示选定区域的实时内容

### 工具栏说明

| 图标 | 功能 | 说明 |
|------|------|------|
| 📐 | 标尺 | 显示/隐藏水平和垂直标尺 |
| 📏 | 辅助线 | 启用/禁用辅助线功能（需先启用标尺） |
| 🎨 | 颜色选择 | 自定义辅助线和网格颜色 |
| 📊 | 线宽 | 调整辅助线和网格的线宽 |
| 🔲 | 网格 | 显示/隐藏网格 |
| 🔍 | 指示器 | 显示/隐藏缩放指示器 |
| 📌 | 置顶 | 将窗口置顶/取消置顶 |

### 高级技巧

- **创建辅助线** - 从水平标尺向下拖拽创建水平辅助线，从垂直标尺向右拖拽创建垂直辅助线
- **调整辅助线** - 点击选中辅助线后可以拖拽调整位置，点击删除按钮移除
- **缩放控制** - 使用鼠标滚轮或捏合手势进行缩放
- **快速关闭** - 在选择模式下按 ESC 键退出
